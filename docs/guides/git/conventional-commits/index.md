# Conventional Commits (index)

## When to read this

- You want consistent commit messages that are easy to scan and automate.

## Files

- `git/conventional-commits/overview.md`: what Conventional Commits are and why use them
- `git/conventional-commits/format.md`: message structure
- `git/conventional-commits/types.md`: common types and when to use them
- `git/conventional-commits/breaking-changes.md`: how to mark breaking changes (`!` and `BREAKING CHANGE:`)
- `git/conventional-commits/examples.md`: practical examples
- `git/conventional-commits/faq.md`: common questions
