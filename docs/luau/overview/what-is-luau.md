# What is Luau?

## When to read this

- You need a quick description of what Luau is and what it’s optimized for.
- You need a reminder of where Luau intentionally diverges from Lua.

## Key points

- Luau is a Lua 5.1–derived scripting language with a focus on performance, sandboxing, and gradual typing.
- Luau supports optional type annotations and a static analyzer (`luau-analyze` in many distributions).
- Some Lua facilities that can weaken sandboxing (e.g. file/system access) are removed or reduced in common embedders.
